# Server-Radis
В проекте используется сервер "TomKat".
Все зависимости прописаны в pom.xml
В web.xml - прописаны сервлены.

Программа состоит из пкетов: Client, servlet, servletList, servletMap.
На "Томкет" развернуты сервлеты.
В зависимости от GET запроса клиента, его код находится в директории Client, этот запрос отправляется на один из сервлетов.

Код клиета разбит на методы: requestMethod() и startGet(String urlIn)
Первый - формирует гет-запрос вида. 
  TYPE
  COMMAND key value
  
  пример: STRING
          SET key1 value1   или  GET key1
         
  ghbvth 2: MAP
            SET key1 keyMap1:value
            
В итоге запрос имеет вид:  http://localhost:8888/servlet?command=GET&key=key1&value=value1&list=0
Здесь: Если STRING, то после http://localhost:8888/servlet
       Если MAP, то servletMap,  Если LIST - servletList  Таким образом каждый тип данных направляется в свой сервис.
       
       Далее знак ? и command=команда, то есть SET, GET, DEL или KEYS * .
       &key=  &value =  &list=
       В key и value попадают данные (ключ и значение). 
       В list- если данные не спиок, то list=0, иначе он будет равен элементам списка разделенные прямой линией
       Пример: http://localhost:8888/servlet?command=GET&key=key1&value=list1&list=|list2|list3|list4
       
Второй метод - отправляет запрос на сервлет и получает от него ответ, выводя результат в терминал.

Код сервлетов примерно одинаков, за исключением некоторой реализации, в зависимости от специфики принятого контента.
В методе doGet() - Читается запрос и разбивается на составляющий. В зависимости от ниx и выполняется логика.
Ответ формируется в виде строки и передается обратно клиенту.
